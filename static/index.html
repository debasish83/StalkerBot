stalkerbot

Reveal how a Hacker News user feels based on their comments

Stalking... This will take a minute.


What is going on here?

Find and scrape top-level HN comments posted by specified user,
Use DiffBot API to extract subject and tags from corresponding article submissions,
Use EffectCheck API to analyze sentiments from user's comments,
Display an amusing and arguably random view of the user's sentiments.
Written with Python, JavaScript, and JQuery. Sorry for the slow CGI method.
